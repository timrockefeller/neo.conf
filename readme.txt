Neovim 0.8 personal configuation for kitekii(timrockefeller).


## enlighten shortcuts

`<leader>c`: buffer delete
`<leader>w`: write
`<leader>q`: quit

## warp

`<leader>f`: hop by word
`<leader>F`: hop by line
`<leader>g`: grep
`<C-e>`: file tree
`<C-p>`: jump to file

## window

`<C-Tab>`: `:bn`
`F`: v-split
`f`: switch window
