Neovim 0.8 personal configuation for kitekii(timrockefeller).

Using **Packer** to manage plugins, and **Mason** to manage language servers.

## Setup

```
:PackerInstall
:Mason
```

## enlighten shortcuts

`<leader>c`: buffer delete
`<leader>w`: write
`<leader>q`: quit

## warp

`<leader>f`: hop by word
`<leader>F`: hop by line
`<leader>g`: grep
`<C-e>`: file tree
`<C-p>`: jump to file

## window

`<C-Tab>`: `:bn`
`F`: v-split
`f`: switch window

## Terminal

`<F7>`: New Terminal
`<F12>`: Toggle between code and terminal
